# Product Overview

This is an academic research project for a Master's degree in Artificial Intelligence (Fundamentos de IA). The project develops an intelligent agent based on Semantic Networks for music recommendation.

## Core Concept

An AI agent that uses semantic networks (graph-based knowledge representation with typed nodes and edges) to recommend music by capturing deep semantic relationships between musical entities — genres, artists, songs, moods, instruments, and eras.

## Key Differentiator

Unlike traditional collaborative or content-based filtering, this approach leverages semantic inference over a knowledge graph to produce explainable recommendations with traceable reasoning paths.

## Project Language

All documentation, requirements, and academic writing are in **Spanish**. Code comments and variable names may use English. When generating or editing documents, default to Spanish unless the file is clearly in English.
